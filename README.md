# ❤️ Interactive Valentine's Landing Page

A modern and responsive Valentine's Day themed landing page built using **HTML5, CSS3, and JavaScript**. The website features a beautiful user interface, smooth animations, an animated background with flowing hearts, and interactive elements to create an engaging user experience.

---

## ✨ Features

- ❤️ Modern and attractive UI design
- 🌹 Animated background with flowing hearts
- ✨ Smooth CSS animations and transitions
- 📱 Fully responsive design
- 💌 Interactive Valentine's themed layout
- 🎨 Clean and elegant interface
- 🖱️ Hover effects and interactive buttons
- ⚡ Fast loading and optimized performance
- 🌐 Cross-browser compatibility
- 📂 Well-structured and maintainable code

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
